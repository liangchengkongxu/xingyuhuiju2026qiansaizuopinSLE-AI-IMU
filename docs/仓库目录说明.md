# 仓库目录说明

## 顶层

| 路径 | 类型 | 说明 |
|------|------|------|
| `README.md` | 文档 | 项目总览、协议摘要、快速导航 |
| `docs/` | 文档 | 联调指南、开发历程、本索引 |
| `bs20/` | 源码 | BS20 拍柄固件（可独立集成进 HiSilicon SDK） |
| `ss928/` | 文档+参考 | SS928 主控对接说明与 IMU 解析参考代码 |
| `.cursor/skills/` | 工具 | Cursor Agent Skill（拍柄开发辅助） |

## bs20/ — 拍柄固件

```
bs20/
├── README.md
├── application/paibing/
│   ├── common/          # IMU 驱动、AHRS、100ms 主循环
│   ├── sle/             # 星闪非连接扫播（量产）
│   └── ble/             # 蓝牙 GATT Notify（调试）
├── tools/               # build_paibing_sle.sh / build_paibing_ble.sh
├── docs/
│   ├── 主控对接说明.md   # ★ 主控必读协议
│   ├── 固件变更记录.md
│   └── 代码文件索引.md
└── integration/         # 合入 fbb_bs2x 的 overlay 与说明
```

**不包含**：`*.fwpkg` 烧录包（本地编译生成，见 `.gitignore`）。

## ss928/ — 主控侧

```
ss928/
├── README.md
├── docs/
│   ├── 主控待办清单.md
│   ├── 部署与运行.md
│   ├── IMU解析与去重实现.md
│   └── 关键源码索引.md
└── reference/ws73/      # 与量产对齐的解析/桥接参考片段
    ├── sle_imu_bridge.sh
    └── sle_seek_print_all/
        ├── sle_seek_print_client.c
        └── sle_imu_adv.h
```

完整 SS928 Qt 面板工程体积大且依赖 MPP SDK，未整包放入本仓；联调以 `reference/` + 文档为准。

## 推荐阅读顺序

1. 根目录 `README.md` — 建立全局认识  
2. `bs20/docs/主控对接说明.md` — 弄清数据从哪来、长什么样  
3. `ss928/docs/主控待办清单.md` — 主控要改什么  
4. `docs/联调快速指南.md` — 实际上板验证  
5. `docs/开发过程与代码要点.md` — 答辩与维护
