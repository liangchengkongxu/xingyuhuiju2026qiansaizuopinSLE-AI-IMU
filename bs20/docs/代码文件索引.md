# 代码文件索引

快速定位拍柄固件各模块源码。

## common/ — 共用

| 文件 | 职责 |
|------|------|
| `paibing_imu.c` | MPU9250 驱动、校准、Mahony AHRS、100ms 主循环 |
| `paibing_imu.h` | 对外 API：`paibing_board_init`, `paibing_imu_run` |
| `paibing_transport.h` | 传输抽象：`init` + `push_sensor` |
| `main.c` | 芯片入口 → `main_init()` |
| `startup.S` | 启动汇编 |

## sle/ — 星闪（量产）

| 文件 | 职责 |
|------|------|
| `paibing_app.c` | 星闪版 `app_main`，注册 transport |
| `paibing_sle_server_adv.c` | **核心**：广播参数、**ASCII @ 行**、异步 restart 刷新 |
| `paibing_sle_server.c` | SLE 服务、`sle_customize_max_pwr` |
| `mac_config.h` | MAC、扫播开关、设备号 |
| `paibing_multi_mode.c` | 多人模式 A1/A2 指令（默认关闭） |

## ble/ — 蓝牙（调试）

| 文件 | 职责 |
|------|------|
| `paibing_app.c` | 蓝牙版 `app_main` |
| `paibing_ble_uart.c` | 连接后 Notify ASCII 九轴 |
| `paibing_ble_sensor_adv.c` | BLE 传感器广播（二进制，备用） |
| `mac_config.h` | 蓝牙 MAC 配置 |

## tools/

| 脚本 | 用途 |
|------|------|
| `build_paibing_sle.sh [MAC末字节]` | 编译星闪 + 复制 all.fwpkg |
| `build_paibing_ble.sh` | 编译蓝牙版 |
| `setup_paibing_ble_prebuild.sh` | BLE target 预编译库链接 |

## integration/

| 路径 | 用途 |
|------|------|
| `sdk_overlay/` | 合并进 fbb_bs2x 的 CMakeLists、config.py 等 |
| `SDK集成说明.md` | 逐步集成指南 |
| `app_os_init.patch` | app_main 弱符号参考 diff |
