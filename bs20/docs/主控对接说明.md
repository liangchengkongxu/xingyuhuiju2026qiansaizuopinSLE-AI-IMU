# 主控对接说明

本文描述 SS928 主控如何接收并解析拍柄 IMU 数据。

---

## 1. 设备识别

| 字段 | 值 |
|------|-----|
| 设备名 | `paibing_imu` |
| MAC 前缀 | `cc:ad:c9:00:22:xx`（末字节 01～06 区分多把拍柄） |
| 上报周期 | 100 ms |

---

## 2. 星闪版（推荐多设备场景）

### 2.1 工作模式

- 模式：`SLE_ANNOUNCE_MODE_NONCONN_SCANABLE`（不可连接、可扫描）
- **不需要建立连接**
- 支持 40+ 把拍柄同时扫播

### 2.2 广播包结构（当前固件）

| 包类型 | 内容 |
|--------|------|
| **ADV** | Discovery + Access + **Manufacturer 0xFF（22 字节二进制）** |
| **Scan Response** | TX Power + 设备名 + **Manufacturer 0xFF（同一份 22 字节）** |

> 两路都带 IMU，主控解析 **ADV 或 Scan Response 任一** 即可。扫到名字说明 Scan Response 收到了；若只有名无数据，检查是否只解析了名字 AD 而漏了解 0xFF。

### 2.3 22 字节二进制帧

```
EB 1A 02 DEV  uptime(2)  ax(2) ay(2) az(2)  gx(2) gy(2)  roll(2) pitch(2)  mag(2)
```

| 偏移 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 0-1 | 厂商 ID | — | 固定 `EB 1A` |
| 2 | 协议 | u8 | 传感器数据 = `0x02` |
| 3 | 设备号 | u8 | 默认 `0x01`，与 MAC 末字节可对应 |
| 4-5 | uptime | u16 LE | 开机 ms 低 16 位 |
| 6-11 | ax,ay,az | i16 LE | 加速度，值×10 = mg |
| 12-15 | gx,gy | i16 LE | 陀螺 dps（无 gz） |
| 16-19 | roll,pitch | i16 LE | 角度×10 = 度 |
| 20-21 | magnitude | u16 LE | 力度×100 |

### 2.4 解析步骤

1. 开启星闪扫描
2. 过滤 `paibing_imu` 或 MAC 前缀
3. 在 ADV **或** Scan Response 原始数据中找 AD：`len | type=0xFF | payload`
4. payload 至少 22 字节，校验头 `EB 1A 02`
5. 按上表小端解析

### 2.5 参考代码（C）

```c
typedef struct {
    uint16_t uptime_ms;
    int16_t ax, ay, az;      /* mg/10 */
    int16_t gx, gy;          /* dps */
    int16_t roll, pitch;     /* deg*10 */
    uint16_t mag;              /* *100 */
} paibing_sensor_frame_t;

int paibing_parse_mfg22(const uint8_t *p, paibing_sensor_frame_t *out)
{
    if (p[0] != 0xEB || p[1] != 0x1A || p[2] != 0x02) return -1;
    out->uptime_ms = p[4] | (p[5] << 8);
    out->ax = (int16_t)(p[6] | (p[7] << 8));
    out->ay = (int16_t)(p[8] | (p[9] << 8));
    out->az = (int16_t)(p[10] | (p[11] << 8));
    out->gx = (int16_t)(p[12] | (p[13] << 8));
    out->gy = (int16_t)(p[14] | (p[15] << 8));
    out->roll = (int16_t)(p[16] | (p[17] << 8));
    out->pitch = (int16_t)(p[18] | (p[19] << 8));
    out->mag = p[20] | (p[21] << 8);
    return 0;
}
```

## 3. 蓝牙版

- 可连接广播，连接后 **GATT Notify**
- 数据为 **ASCII 行**（非 22 字节二进制）：

```
@185230,A-13,+1,+98,G-2,-1,R+123,P-456,M102\n
```

| 字段 | 缩放 |
|------|------|
| A 三轴 | ×10 → mg |
| G 两轴 | dps |
| R/P | ÷10 → 度 |
| M | ÷100 → g |

---

## 4. RSSI 说明

主控显示的「80 多」通常是 **-80 dBm**（弱信号），不是发射功率 80 dBm。拍柄实际发射约 **6～8 dBm**。
