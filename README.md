# 星羽汇聚 · 多模态星闪羽毛球智能训练系统

> **2026 全国大学生嵌入式芯片与系统设计竞赛 · 海思 SS928 端侧智能赛题**

基于 **海思 SS928 边缘计算 + 端侧视觉 AI + 星闪 SLE 高并发通信**，面向校园体育教学与专业训练的羽毛球智能分析系统。  
从「经验教学」走向「数据化科学训练」。

---

## 项目简介

传统羽毛球训练高度依赖教练主观判断，缺乏客观、可量化的动作数据支撑。本作品针对这一痛点，设计了一套 **双模态自适应** 的智能训练系统：

- **单人对打模式**：视觉 AI + 拍柄九轴 IMU **多模态融合**，精细评估击球类型、发力轨迹与动作质量  
- **班级多人模式**：星闪 SLE 高并发接入 **20+ 拍柄节点**，纯 IMU 轻量解算，满足大班额同步训练

系统以 SS928 为主控、10.1 英寸触摸屏为交互终端，实现毫秒级反馈与可视化训练报表，适用于体育中考、专业队选材与校园日常教学。

---

## 核心亮点

| 亮点 | 说明 |
|------|------|
| **双模态自适应** | 单人模式「视觉 + IMU」深度融合；多人模式「纯 IMU」轻量解算，按场景自动切换 |
| **星闪 SLE 高并发** | 非连接扫播替代传统蓝牙，支持 20+ 设备同时在线，解决班级训练信道拥堵 |
| **端侧 AI 零感延迟** | YOLOv8 等模型下沉 SS928 本地 NPU，端到端反馈 **< 50 ms**，数据不出端 |
| **触摸屏一体化交互** | Qt UI 支持模式切换、实时看板、历史报表，开箱即用 |

## 两大业务模式

### 单人对打 · 多模态融合

面向个人精细化训练与专业复盘。

```
  摄像头 ──► SS928 端侧 AI（姿态/动作识别）
                │
  拍柄 IMU ─────┤──► 多模态融合 ──► 击球类型 · 挥拍速度 · 动作评分
                │
           触摸屏 ◄── 实时指导与改进建议
```

- 识别高远球、杀球、吊球等击球类型  
- 计算挥拍速度、拍面角度与动作规范度  
- 融合准确率 **> 85%**

### 班级多人 · 星闪高并发

面向学校大班额体育课与体测场景。

```
  拍柄 #1 ─┐
  拍柄 #2 ─┤
  拍柄 #3 ─┼── 星闪 SLE 广播 ──► SS928 并发接收 ──► 全班训练看板
    …      │
  拍柄 #20 ┘
```

- 关闭视觉分析，仅依赖九轴 IMU，降低算力开销  
- 支持 **20+ 节点** 同时在线  
- IMU 采样 **100 Hz**，满足同步训练与数据汇总

## 创新点

1. **场景自适应双模态架构** — 首创「单人视觉融合 + 多人纯 IMU」切换机制，平衡高精度分析与大规模并发  
2. **星闪 SLE 赋能校园体育** — 将星闪高并发能力引入班级训练，攻克高密度设备接入瓶颈  
3. **端侧多模态数据融合** — 在嵌入式端实现视觉与 IMU 的实时时空对齐，克服单一传感器盲区与漂移
