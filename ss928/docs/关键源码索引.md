# 关键源码索引

完整 SS928 工程在队伍内部 SDK 工作区（如 `SS928V100_SDK_V2.0.2.2/version3.0/`）。下表便于从本 GitHub 仓跳转到对应模块。

---

## 星闪 IMU 接收

| 文件 | 职责 |
|------|------|
| `ws73/sle_seek_print_all/sle_seek_print_client.c` | ★ TLV 解析、ASCII 优先、MAC+uptime 去重 |
| `ws73/sle_seek_print_all/sle_imu_adv.h` | 协议常量（ASCII 为主，EB 1A 回退） |
| `ws73/sle_imu_bridge.sh` | 守护 `sle_seek_print_all` → `/tmp/sle_imu_lines` |
| `ws73/ws73_common.sh` | 加载 `sle_soc.ko`、WiFi 共存 |
| `scripts/deploy_ws73.sh` | 编译部署星闪工具 |

本仓副本 → [`../reference/ws73/`](../reference/ws73/)

## Qt IMU 业务

| 文件 | 职责 |
|------|------|
| `src/sle_imu_service.cpp` | 读日志、去重、CNN/规则击球、发信号 |
| `src/sle_imu_service.h` | |
| `src/imu_swing_detector.cpp` | 规则 FSM 挥拍检测、速度/力度估算 |
| `src/imu_cnn_classifier.cpp` | 1D CNN 峰值触发 + 动作分类 |
| `src/imucnn/badminton_npu_infer.*` | NPU 推理封装 |
| `models/badminton_npu.om` | 训练好的 OM 模型 |

## 与 bs20 对应关系

| bs20 | SS928 |
|------|-------|
