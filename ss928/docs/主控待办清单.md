# SS928 主控待办清单

拍柄（BS20）固件已更新为 **2026-07 ASCII 扫播版**。主控按此清单对接。

**详细协议** → [`../../bs20/docs/主控对接说明.md`](../../bs20/docs/主控对接说明.md)  
**实现说明** → [IMU解析与去重实现.md](IMU解析与去重实现.md)

## 必做项

| # | 事项 | 状态 | 说明 |
|---|------|------|------|
| 1 | **ASCII 行解析** `@uptime,A...,G...,R...,P...,M...\n` | ✅ | `reference/ws73/sle_seek_print_client.c` |
| 2 | **停止 `EB 1A 02` 二进制默认解析** | ✅ | 仅作旧固件回退 |
| 3 | **ADV + Scan Response 取 0xFF** | ✅ | 每次扫描报告遍历 TLV |
| 4 | **MAC + `@uptime_ms` 去重** | ✅ | C 层 + Qt `sle_imu_service` 双层 |
| 5 | **验证 10Hz 有效样本** | 🔧 联调 | 去重后 uptime ≈ 每 100ms +1 |

---

## 验收标准

1. `tail -f /tmp/sle_imu_lines` 中同一 MAC 的 uptime **不重复刷屏**
2. 相邻有效帧 uptime 差约 **80～120ms**（允许偶尔丢包 >200ms）
3. 挥拍时 `G`/`R`/`P` 有明显变化，非长期全 0
4. 数值与拍柄 UART 打印量级一致（A×10=mg）

### 参考日志

@11099,A-75,...   ← 扫描重复，去重丢弃

## 建议项

- 扫描占空比尽量大（`WIDGET_WS73_BSLE_MAX_COEX=1` 提高共存优先级）
- 多设备时按 MAC 独立去重表（已实现 16 槽 LRU）
- RSSI 按 **负数 dBm** 理解（-80 = 偏弱）
- 联调烧录：`paibing_sle_mac01_all.fwpkg`
