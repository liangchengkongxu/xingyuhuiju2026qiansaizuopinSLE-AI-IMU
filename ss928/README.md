# SS928 主控端（端侧智能）

本目录用于存放 **SS928** 端侧智能主控代码，包括：

- 星闪（SLE）扫描与 IMU 数据解析
- 端侧 AI 推理与训练业务逻辑
- 与拍柄（BS20）的联调接口

## 与 BS20 对接

拍柄广播格式见 [`../bs20/docs/主控对接说明.md`](../bs20/docs/主控对接说明.md)。

- 设备名：`paibing_imu`
- MAC 前缀：`cc:ad:c9:00:22:xx`
- 载荷：ADV / Scan Response 中 `0xFF` 厂商域，22 字节 `EB 1A 02 ...`
